## â“ 9. FAQ

**Q: Apakah data saya aman?**  
A: Ya! Data disimpan lokal di SQLite, tidak dikirim ke server manapun.

**Q: Bisa akses dari device lain?**  
A: Untuk saat ini hanya lokal. Update mendatang akan add cloud sync.

**Q: Bagaimana cara delete semua data?**  
A: `php artisan migrate:refresh --seed` untuk reset. Atau delete `database/database.sqlite` dan migration ulang.

**Q: Bisa add users/teams?**  
A: Feature ini planned untuk update berikutnya.

**Q: Gimana kalau app crash?**  
A: Data aman di database. Restart app, data tetap ada.

**Q: Bisa customize theme colors?**  
A: Edit `tailwind.config.js` dan `resources/css/app.css` sesuai kebutuhan.

### Problem: Task tidak muncul
**Solution:**
- Refresh halaman (Ctrl + R)
- Clear browser cache (Ctrl + Shift + R)
- Check browser console (F12) untuk error

**Solution:**
